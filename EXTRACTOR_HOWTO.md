## Troubleshooting

### Extractor fails?
```bash
# Check logs
cat terraform/extractor-run-*.log

# Or on EC2
ssh -i ~/.ssh/id_rsa ubuntu@54.220.110.151
tail -f ~/ido-esperanto-extractor/logs/*.log
```

### Disk full?
```bash
cd ~/apertium-dev/terraform
./cleanup_ec2_extractor.sh
```

### Instance stuck?
```bash
cd ~/apertium-dev/terraform
./start_stop.sh stop
./start_stop.sh start
```
